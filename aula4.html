Aula 4: JavaScript

No inicio da internet, a unica coisa possivel era navegar atraves de links, e enviar informações por formularios

A partir disso, se notou a necessidade de mais interação, é para isso que o JavaScript existe. A Netscape, criadora do navegador mais popular do inicio dos anos 90 de mesmo nome, criou o Livescript, uma linguagem simples que permitia execução de scrips no navegador.

Aproveitando o sucesso do Java, rebatizaram o Livescript de JavaScript.

A vice lider Microsoft, adiciounou ao Internet Explorer o suporte a scripts escritos em VBScript e cripo a sua versão do JavaScript, o JScript.

Hoje, o JavaScript é a linguagem de programação mais popular do mundo, épossivel criar excelentes aplicaçãos, exemplos é Gmail, Google Maps e Docs.

Caracteristicas da Lingugem

JavaScript é uma linguagem de Scripting.

Scripting

Linguagem de scripting é comumente definida como uma linguagem de programação que permite ao programador controlar uma ou mais aplicações de terceiros.

O JS permite controlar alguns comportamentos do navegador atraves de trechos de codigo que são enviados na pagina HTML.

O JS tem grade tolerancia a erros, pois conversões autmaticas são realizadas durne operação, e que se não conhecermos bem esse mecanismo pode ser fontes de bugs tambem.

O script do programador é enviado com o HTML, e para diferenciar HTML do JS, o script é colocado dentro de uma tag <script> e o navegador sabe que tudo que estiver dentro dessa tag é JS, e não HTML.

A Tag Script

A tag <script> pode ser declarada dentro da tag <head>, no entanto o script vai travar a pagina, pois será lido primeiro, então se tiver alguma coisa ali mais complexa vai travar ali até terminar de ler, então podemos colocar no <body> do documento HTML, no final, assim a pagina é executada e depois execulta o script.

JS em arquivo externo

Utilizando o atributo src na tag <script>, podemos carregar o código JavaScript de um arquivo externo.

Por exemplo: <script type="text/javascript" src="meu_script.js"></script>
