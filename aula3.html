AULA 3: CSS avançado

Com o avanço do html e css, foi possivel fazer coisas que antes eram impossiveis, que antes eram feitas com imagem ou até com o uso de javascript, por isso, veremos algumas das tecnicas mais avançadas de css

Seletores avançados

Vimos seletores mais avançdos como o de atributo, mas agora veremos outros

Seletores irmãos

<article>
    <h1>Titulo</h1>
    <p>Inicio</p>


    <h2>Subtitulo</h2>
    <p>Texto</p>
    <p>Mais texto</p>
</article>

Se quisermos estiliza todos os paragrafos apos o subtitulo?

Usamos o seletor de irmão (siblings) (~).

Ele serve para selecionar elementos que são irmãos de um elemento específico.

Ficando:

h2~p{
    color: red;
}

Seletores irmãos adjacente

O seletor de irmãos adjacente (+) seleciona o elemento imediatamente seguinte ao elemento especificado.

Ficando:

h2+p{
    color: red;
}

Assim apenas o "texto" é aplicado a cor red, o "mais texto" não será afetado.

Seletor de filho direto

O seletor de filho direto (>) serve para selecionar elementos que estão imediatamente dentro de outro, sem considerar os "netos" ou elementos em níveis mais profundos da estrutura.

Imagine o seguinte cenário em HTML:

<article>
    <h1>Título principal</h1>

    <section>
        <h1>Título da seção</h1>
    </section>
</article>

Se você quiser alterar a cor apenas do título principal, usar um seletor de hierarquia comum (com espaço) não vai funcionar, pois ele pintaria ambos os títulos de azul:

/* Errado: isso vai selecionar TODOS os h1 dentro do article */
article h1 {
    color: blue;
}

Para resolver isso, usamos o operador maior que (>). Ele garante que apenas o h1 que é filho imediato do article receba o estilo:

/* Correto: seleciona apenas o h1 que é filho direto */
article > h1 {
    color: blue;
}

Seletor de negação

O seletor de negação (:not) serve para aplicar estilos a elementos que não possuem uma determinada característica, como uma classe ou um ID específico.

Considere esta lista de parágrafos:

<p>Texto</p>
<p>Outro texto</p>
<p class="especial">Texto especial</p>
<p>Mais texto</p>

Se a intenção for deixar todos os parágrafos cinzas, exceto aquele que foi marcado com a classe "especial", usamos a pseudo-classe :not() passando o alvo da exceção entre os parênteses:

/* Seleciona todos os parágrafos que NÃO têm a classe .especial */
p:not(.especial) {
    color: gray;
}

Pseudo-classes

Imagine que temos uma lista simples no HTML e queremos estilizar elementos específicos dela — por exemplo, deixar o primeiro item vermelho e o último azul:

<ul>
    <li>Primeiro item</li>
    <li>Segundo item</li>
    <li>Terceiro item</li>
    <li>Quarto item</li>
</ul>

Uma solução comum seria adicionar classes manuais no HTML. Porém, isso exige alteração estrutural e manutenção constante se a ordem dos itens mudar. Para resolver isso de forma inteligente, o CSS oferece as pseudo-classes.

Elas funcionam como classes pré-definidas pelo próprio navegador, permitindo selecionar elementos com base em sua posição ou estado sem mexer no HTML. Para diferenciá-las de uma classe comum, usamos o caractere de dois pontos (:) em vez do ponto.

/* Seleciona o primeiro li filho */
li:first-child {
    color: red;
}

/* Seleciona o último li filho */
li:last-child {
    color: blue;
}

O seletor nth-child

Para um controle ainda mais flexível, utilizamos o seletor :nth-child(), que recebe um índice ou uma expressão matemática entre os parênteses para definir quais elementos serão afetados.

Podemos selecionar um elemento exato informando o seu número:

/* Seleciona especificamente o terceiro item */
li:nth-child(3) {
    color: yellow;
}

Também é possível criar um padrão alternado (como uma lista zebrada) utilizando expressões aritméticas para atingir índices pares e ímpares:

/* Seleciona os elementos pares (2, 4, 6...) */
li:nth-child(2n) {
    color: green;
}

/* Seleciona os elementos ímpares (1, 3, 5...) */
li:nth-child(2n+1) {
    color: blue;
}

Pseudo-classes de estado

Outra utilidade importante das pseudo-classes é reagir às ações do usuário em tempo real, capturando o estado atual de um elemento (como passar o mouse por cima ou clicar) de forma nativa, sem depender de scripts externos.

A pseudo-classe :hover é acionada no exato momento em que o ponteiro do mouse é posicionado sobre o elemento:

/* Altera o fundo quando o usuário passa o mouse pelo link */
a:hover {
    background-color: #FF00FF;
}

Especificamente para links (âncoras), existem outras pseudo-classes fundamentais para gerenciar o comportamento visual de navegação:

/* Seleciona links padrão que possuem o atributo href */
a:link {
    background-color: #FF0000;
}

/* Seleciona links que o usuário já visitou no histórico */
a:visited {
    background-color: #00FF00;
}

/* Seleciona o link durante o clique ativo do mouse */
a:active {
    background-color: #0000FF;
}

Pseudo-elementos

Enquanto as pseudo-classes selecionam elementos que já existem com base em sua posição ou estado, os pseudo-elementos permitem estilizar partes específicas do conteúdo de um elemento que sequer possuem uma tag própria no HTML (ou até mesmo gerar novos conteúdos visuais).

Para diferenciar visualmente os pseudo-elementos das pseudo-classes no CSS moderno, utilizamos a sintaxe de dois pontos duplos (::).

Imagine que você queira criar um efeito de revista, aumentando o tamanho apenas da primeira letra de um parágrafo comum:

<p>A Caelum tem os melhores cursos!</p>

Em vez de poluir o HTML envolvendo a letra "A" com uma tag desnecessária, usamos o pseudo-elemento ::first-letter diretamente no CSS:

/* Seleciona e estiliza apenas a primeira letra do parágrafo */
p::first-letter {
    font-size: 200%;
}

Da mesma forma, se precisarmos aplicar um estilo exclusivo para a primeira linha de um bloco de texto (independentemente do tamanho da tela do usuário), podemos usar o ::first-line:

/* Deixa apenas a primeira linha do parágrafo em itálico */
p::first-line {
    font-style: italic;
}

Inserindo novos conteúdos via CSS

Existem dois pseudo-elementos muito poderosos chamados ::before e ::after. Eles servem para injetar conteúdo puramente estético antes ou depois do conteúdo real de um elemento, mantendo o HTML limpo e focado apenas no significado do texto (sem afetar leitores de tela ou mecanismos de busca).

Considere esta lista simples de links:

<a href="#">Link 1</a>
<a href="#">Link 2</a>

Se quisermos cercar esses links com colchetes apenas por uma escolha de design, usamos a propriedade content dentro dos pseudo-elementos:

/* Adiciona um colchete antes do texto do link */
a::before {
    content: '[ ';
}

/* Adiciona um colchete depois do texto do link */
a::after {
    content: ' ]';
}

Podemos ir além e combinar pseudo-elementos com seletores de atributo. Por exemplo, para identificar de forma automática e visual todos os links externos da página que começam com "http://", adicionando o aviso "(externo)" após o texto deles:

/* Identifica links externos e insere um aviso ao final */
a[href^="http://"]::after {
    content: ' (externo)';
}
