HTML

Oque é html?

Esqueleto do navegador, sua estrutura de forma bruta, sem a estetica. De forma mais formal, é a marcação do conteudo na pagina WEB.

Tags

Sintexe do html é feita na forma <_>. Vai nomes especificos dentro daqueles sinais. <NOMES_ESPECIFICOS> são chamados de tags. Por exemplo: <h1>, é uma tag, e essa é a tag para titulos principais da pagina.

Muitas tags tem conteudo. Por exeplo: <h1>TITULO: HTML </h1>. Note que usamos o contrabarra+NOMEE_DA_TAG para fechar o conteudo.

Atributo

Muitas tags tem atributos. Ele vem na sua definição ficando <NOME_DA_TAG ATRIBUTO=VALOR>. Por exemplo:

<img src="endereço_da_imagem">

<tag atributo="valor">

Veja que não tem conteudo em si, então não precisa de tag de fechamento (/mome_da_tag)

Estilizando com CSS

Quando colocamos uma tag, e ela aparece no navegador, ela tem um estilo padrão, como tamanho, fonte e cor. Mas podemos customizar esse estilo com css.

Antigamente usava-se no propeio html. Ex:<h1><font color="red">Texto</font></h1>

Assim ficaria vermelho o texto. Mas não usamos mais tags de estilo. Usamos o CSS.

CSS=linguagem separada do html para cuidar da estilização da pagina.

Sintaxe e Inclusão do CSS

Propriedade e valores separados por um sinal de dois pontos ":", e cada conjunto de propriedade+valor separam-se de outra propriedade+valor com um ponto e virgula ";"

{color: blue; background-color: yellow;}

Nesse exemplo as tags que forem encapsulada com esse css vai ficar azul, com fundo amarelo

Podemos usar o css no html tambem. Mas voce não acabou de dizer que ele existe justamente para sair do html? Calma, antes do css usavamos tags de estilo, não usamos mais tags de estilo, usamos o css, que tem uma sintese especifica, e que pode ser usada no htl ou em um arquivo separado (melhor pratica)

Ex de css dentro do html:<p style="color: blue; background-color:yellow;"> Conteudo</p>

Alem disso, outra forma de usar css dentro do html seria, ao inves de tag de estilo (não usadas) e ao inves de integrar a uma tag como vimos ela no <p>, podemos usar uma tag criada para o css: <style>. Note que não é uma tag de estilo (que não deve ser usada), é uma tag propria para o css, que agrupa o css em sua propria sintexe.

Tanto nessa ultima forma quanto css em outro arquivo usamos o seletor css para indicar qual tag vamos estilizar

<style>

p{ color: blue; background-color: yellow; }

</style>

No exemplo acima significa que todas as tag p (p sendo o seletor css) terá esse estilo.

Observação:Essa forma, com a tag style, deve ser colocada no head

Arquivo exteno

Para ter o css em um outro arquivo, devemos conectar o css ao html, para isso usamos a extensão .css, ou seja, seu arquivo css deve ser NOME_DO_ARQUIVO.css

Colocamos um link no head: <link rel=stylesheet" href="estilo.css">

Propriedades tipograficas e fontes

Propriedade font-family

Recebe o valor com ou sem aspas, primeiramente veremos com as familias.

As fontes tem familias

Serif: São chamadas de fontes serifadas, por conta do pequeno ornamento em sua terminação, e costuma ser a padão. As mais usadas é times e times new roman

Sans-serif: Podemos usar sem serifas, como por exemplo Arial, Helvetica.

Monospace: como por exemplo a fonte Courier.

Ex: h1{ font-family: serif; }

Podemos usar a fonte especifica alem da familia

Ex: h1{ font-family: "times", serif; }

Podemos colocar uma segunda opção para o navegador caso ele não tenha dispovinvel a primeira

Ex: h1{ font-family: "times", "times new roman", serif; }

Se não tiver a primeira, verificará a segunda, se não tiver a segunda, diz para ser qualquer uma serif

Propriedade font-style

Define o estilo da fonte. Normal, italic e oblique

Alinhamento e decoração de texto

Propriedade text-align

Alinhamento do texto.

text-align: right = direita

text-align: left = esquerda

text-align: center = centro

text-align: justify = justificado

É possivel configurar outros espaçamentos de texto

p{ line-height: 3px; /* tamanho da altura de cada linha */ letter-spacing: 3px; /* tamanho do espaço entre cada letra */ word-spacing: 5px; /* tamanho do espaço entre cada palavra */ text-indent: 30px; /* tamanho da margem da primeira linha do texto */ }

Cores na web

Podemos definir cor pelo seu nome

h1{ color: red; }

Mas não tem variação só escrevendo o nome, voce que um tom especifico é mais facil usar a composição RGB. 16 milhoes de corem combinando 3 cores bases:Vermelho (red), Verde (Green), Azul (Blue). Enão cada uma dessas tres vai de 0 a 255, por exemplo, amarelo forte = (255, 255, 0)

A sintexe:

h1{ color: rgb(255, 255, 0); }

No entanto, o mas usado é a notação hexadecimal. Usamos o #, depois temos dois caracter para o Red, dois para o Green e dois para o Blue. Com dois caracter em hexadecimal, podemos escrever facilmente esses 255 valores de cada cor, por exemplo: rgb(242, 237, 237), 242=F2, 237=ED, 237=ED, então fica #F2EDED.

Espaçamento, Margem e Dimensões

Padding

ESpaçamento:Para definir espaçamento interno em algum elemento, ou seja, o limite de sua borda ao conteudo

padding-top

padding-right

paddong-bottom

padding-left

Esssas propriedade aplicam uma distancia entre o limite do elemento e seus conteudo acima, direira, abaixo e esquerda. Podemos Podemos definir essas subpropriedades comum unico comando que seja jogado para todos

p{ padding:10px; }

Se passar dois valores, o primeiro sera mandado para cima e para baixo, o segundo para esquerda e direita

Se passar 3 valores, o primeiro sera mandado para cima, o segundo para direita e esquerda, e o terceiropara baixo

Se passar 4 valores, respectivamente: cima, direita, baixo, esquerda

Margin

Margem:Parecido com a anterior, mas vai dar um espaço apos o elemento em si, ou seja, alem dele

As subpropriedade são as mesmas e funciona da mesma forma:

margin-top

margin-right

margin-bottom

margin-left

O margin permite que o navegador defina qual será a dimensão de paddin ou margin conforme o espaço disponivel na tela, usamos o valor "auto"

Para isso é só colocar alto no lugar do valor que voce deseja substituir, exemplo:

p{ margin: 0 auto; }

Então pela regra, temos dois valores, o primeiro seria para cima e baixo, que eu coloquei 0, e o segungo para direita e esquerda, que eu coloquei auto

FLuxo do documento e float

Perceba que o html por padrão deixa todo o conteudo um em baixo do outro, como caixas empilhadas, para sai desse fluxo de documento, podemos usar o float.

.foto-familia { float: left; /* Faz a imagem flutuar para a esquerda */ margin-right: 20px; /* Cria um espaço entre a imagem e o texto ao lado */ margin-bottom: 10px; /* Cria um espaço para o texto não colar embaixo */ width: 200px; /* Define um tamanho para a imagem */ }

Como o elemento com float sai do fluxo normal da página, os elementos que vêm muito depois dele (como um rodapé ou outra seção) podem tentar subir e se misturar de forma bagunçada. Para evitar que isso aconteça, nós usamos uma propriedade chamada clear no elemento que deve ficar obrigatoriamente abaixo de tudo, reiniciando o fluxo normal do HTML.

.elemento-de-baixo { clear: both; /* Impede que este elemento suba para o lado de qualquer float */ }
