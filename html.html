HTML

Oque é html?

Esqueleto do navegador, sua estrutura de forma bruta, sem a estetica. De forma mais formal, é a marcação do conteudo na pagina WEB.

Tags

Sintexe do html é feita na forma <_>. Vai nomes especificos dentro daqueles sinais. <NOMES_ESPECIFICOS> são chamados de tags. Por exemplo: <h1>, é uma tag, e essa é a tag para titulos principais da pagina.

Muitas tags tem conteudo. Por exeplo: <h1>TITULO: HTML </h1>. Note que usamos o contrabarra+NOMEE_DA_TAG para fechar o conteudo.

Atributo

Muitas tags tem atributos. Ele vem na sua definição ficando <NOME_DA_TAG ATRIBUTO=VALOR>. Por exemplo:

<img src="endereço_da_imagem">

<tag atributo="valor">

Veja que não tem conteudo em si, então não precisa de tag de fechamento (/mome_da_tag)

Para ser valido o documento precisa de: <html>, <head>, <body>

Dessa forma, veremos cada uma

<html>

Na estrutura do documento inserimos antes de tudo essa tag: <html>. Dentro dela temos duas tags. Quando tag esta dentro de outra podemos dizer que elas são filhas (internas) e mãe (externa).As duas filhas pode ser chamadas de irmãs.

<html>

<head></head>

<body></body>

</html>

<head>

Contem informações sobre o documento que interessa apenas ao navegador e não ao usuario visitante.

Dentro dela precisa obricatoriamente tem um titulo, e usamos a tag: <title>

Como title esta dentro do head não vai aparecer para o usuario na tela, ela é o titulo do documento, e ficará na barra de titulo no topo do navegador

Para a configuração de idiomas como o portugues que usa caracteres com acento, usamos o encoding ou charset. Fica na tag <meta>. Atualmente usamos o UTF-8, mas ja usamos o latin1

<html>

<head></head>

<metaa charset="utf-8">

<title>TITULO_DO_DOCUMENTO</title>

<body></body>

</html>

<body>

Aqui tudo que estiver será exibido para o usuario visitante.

Deverá ter um ou mais tags ali dentro.

<html>

<head></head>

<metaa charset="utf-8">

<title>TITULO_DO_DOCUMENTO</title>

<body>

<h1>Titulo_principal_que_aparece_para_o_usuario</h1>

</body>

</html>

DOCTYPE

Não é tag, é instruções especiais.

Fala para o navegador qual versão html deve ser usada.

Usamoso <!DOCTYPE html>. Que indica que deve ser usada a versão mais recente, que atualmente é a versão 5.
